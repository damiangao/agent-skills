# OKR Tracker - 目标追踪技能

帮助用户追踪 OKR 进度，设置周期性提醒，生成进度报告。

---

## 🎯 核心功能

| 功能 | 说明 |
|------|------|
| **周期性提醒** | 每日/每周/每月任务提醒 |
| **进度追踪** | 更新 OKR 完成进度 |
| **自动报告** | 月末/季度汇总报告 |
| **快速命令** | 自然语言更新进度 |

---

## 📁 数据存储

### 追踪文件位置

```
~/.openclaw/workspace/okr-tracker.md    # 主追踪文件
~/.openclaw/workspace/memory/           # 每日记录
```

### 数据格式

追踪文件使用 Markdown 格式，包含：
- OKR 目标与关键结果
- 进度状态（复选框 + 百分比）
- 周期性任务清单
- 更新日志

## 🔧 工具调用

### Cron 提醒设置

**每日提醒模板**：
```json
{
  "action": "add",
  "job": {
    "name": "{任务名}",
    "schedule": {
      "kind": "cron",
      "expr": "0 9 * * *",
      "tz": "Asia/Shanghai"
    },
    "sessionTarget": "isolated",
    "wakeMode": "now",
    "payload": {
      "kind": "agentTurn",
      "message": "提醒用户：{提醒内容}",
      "deliver": true,
      "channel": "{当前渠道}",
      "to": "{用户 ID}"
    }
  }
}
```

**每周提醒模板**（每周五晚 8 点）：
```json
{
  "schedule": {
    "kind": "cron",
    "expr": "0 20 * * 5",
    "tz": "Asia/Shanghai"
  }
}
```

**每月提醒模板**（每月 28 号）：
```json
{
  "schedule": {
    "kind": "cron",
    "expr": "0 9 28 * *",
    "tz": "Asia/Shanghai"
  }
}
```

## 📊 进度更新规则

### 自动解析用户输入

| 用户输入 | 解析逻辑 |
|----------|----------|
| "锻炼 3 次" | 提取数字 3，更新 `本周锻炼：3 次` |
| "工作时间 120 小时" | 提取数字 120，更新 `本月工作时间：120/180 小时` |
| "建仓 10 万" | 提取数字 100000，更新 `当前进度：10/50 万` |
| "完成 50%" | 提取 50%，更新对应进度条 |

### 更新文件格式

使用 `edit` 工具精确替换追踪文件中的对应行：

```markdown
### KR1 核心保障
- [ ] 本月工作时间：**120 / 180 小时**（日均≤6 小时）
- [ ] 本周锻炼：`3 次`（目标：每周）
```

## 📅 默认提醒配置

### 建议的周期性任务

| 频率 | 时间 | 内容 | Cron 表达式 |
|------|------|------|-------------|
| 每日 | 21:00 | 工作时间记录 | `0 21 * * *` |
| 周一 | 09:00 | 锻炼计划 | `0 9 * * 1` |
| 周五 | 20:00 | 项目检查 | `0 20 * * 5` |
| 周日 | 21:00 | 投资复盘 | `0 21 * * 0` |
| 每月 28 日 | 09:00 | OKR 复盘 | `0 9 28 * *` |

## ⚠️ 注意事项

1. **时间约束优先**：始终提醒用户 180 小时/月的硬约束
2. **生活第一**：O3 是根基，当 O1/O2 与 O3 冲突时优先保护 O3
3. **输出导向**：鼓励用户记录输出（文章、开源项目）
4. **季度复盘**：每季度末主动提醒深度复盘

## 📖 使用示例

### 示例 1：设置每日提醒

**用户**: 每天早上 9 点提醒我记录工作时间

**AI**: 
1. 调用 cron 工具创建每日 9 点提醒
2. 回复：`⏰ 好的，每天早上 9 点提醒你记录工作时间~`

### 示例 2：更新进度

**用户**: 本周锻炼 3 次

**AI**:
1. 读取 `okr-tracker.md`
2. 更新 `本周锻炼：3 次`
3. 回复：`✅ 已更新：本周锻炼 3 次。继续加油！💪`

### 示例 3：查询进度

**用户**: 我 OKR 进度怎么样

**AI**:
1. 读取 `okr-tracker.md`
2. 汇总各目标进度
3. 回复格式化进度报告

### 示例 4：生成月报

**用户**: 生成 3 月月度报告

**AI**:
1. 读取 `memory/2026-03-*.md` 文件
2. 汇总本月所有记录
3. 生成月度报告并保存到 `memory/2026-03-report.md`
